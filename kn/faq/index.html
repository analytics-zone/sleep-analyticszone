ಪದೇ ಪದೇ ಕೇಳಲಾಗುವ ಪ್ರಶ್ನೆಗಳು

ನಿಮ್ಮ ಆಪಲ್ ಹೆಲ್ತ್ ದತ್ತಾಂಶವನ್ನು ರಫ್ತು ಮಾಡುವ ಮತ್ತು ವಿಶ್ಲೇಷಿಸುವ ಬಗ್ಗೆ ಸಾಮಾನ್ಯ ಪ್ರಶ್ನೆಗಳು

ನನ್ನ ಆಪಲ್ ಹೆಲ್ತ್ ದತ್ತಾಂಶವನ್ನು CSV ಗೆ ರಫ್ತು ಮಾಡುವುದು ಹೇಗೆ?

ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್‌ನೊಂದಿಗೆ, ನಿಮ್ಮ ಆಪಲ್ ಹೆಲ್ತ್ ದತ್ತಾಂಶವನ್ನು CSV ಗೆ ರಫ್ತು ಮಾಡುವುದು ಸರಳವಾಗಿದೆ:

  1. App Store ನಿಂದ ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ
  2. ಅಪ್ಲಿಕೇಶನ್ ತೆರೆಯಿರಿ ಮತ್ತು HealthKit ಅನುಮತಿಗಳನ್ನು ನೀಡಿ
  3. ನೀವು ರಫ್ತು ಮಾಡಲು ಬಯಸುವ ಮೆಟ್ರಿಕ್‌ಗೆ ಹೋಗಿ
  4. Export ಬಟನ್ ಒತ್ತಿ ಮತ್ತು "CSV" ಸ್ವರೂಪವನ್ನು ಆರಿಸಿ
  5. ನಿಮ್ಮ ದಿನಾಂಕ ಶ್ರೇಣಿ ಮತ್ತು ನಿರ್ದಿಷ್ಟ ದತ್ತಾಂಶ ಅಂಶಗಳನ್ನು ಆಯ್ಕೆಮಾಡಿ
  6. CSV ಫೈಲ್ ಅನ್ನು ಹಂಚಿಕೊಳ್ಳಿ ಅಥವಾ ಉಳಿಸಿ

CSV ಫೈಲ್ ಎಕ್ಸೆಲ್, ಗೂಗಲ್ ಶೀಟ್ಸ್ ಅಥವಾ ಯಾವುದೇ ಸ್ಪ್ರೆಡ್‌ಶೀಟ್ ಅಪ್ಲಿಕೇಶನ್‌ನಲ್ಲಿ ತೆರೆಯುತ್ತದೆ. ಆಪಲ್‌ನ ಸ್ಥಳೀಯ XML ರಫ್ತಿನಂತಲ್ಲದೆ, ನಮ್ಮ CSV ಫೈಲ್‌ಗಳು ವಿಶ್ಲೇಷಣೆಗಾಗಿ ತಕ್ಷಣವೇ ಸಿದ್ಧವಾಗಿರುತ್ತವೆ.

ನನ್ನ HealthKit ದತ್ತಾಂಶವನ್ನು ನನ್ನ ವೈದ್ಯರೊಂದಿಗೆ ಹಂಚಿಕೊಳ್ಳಬಹುದೇ?

ಹೌದು! ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ನಿಮ್ಮ ಆರೋಗ್ಯ ದತ್ತಾಂಶವನ್ನು ವೈದ್ಯರೊಂದಿಗೆ ಹಂಚಿಕೊಳ್ಳುವುದನ್ನು ಸುಲಭಗೊಳಿಸುತ್ತದೆ. ನೀವು ಈ ಕೆಳಗಿನವುಗಳನ್ನು ಒಳಗೊಂಡಿರುವ ವೃತ್ತಿಪರ PDF ವರದಿಗಳನ್ನು ರಚಿಸಬಹುದು:

  • ನಿಮ್ಮ ನೇಮಕಾತಿಗೆ ಸಂಬಂಧಿಸಿದ ಆಯ್ದ ಆರೋಗ್ಯ ಮೆಟ್ರಿಕ್‌ಗಳು
  • ಕಾಲಾನಂತರದಲ್ಲಿ ಆಗುವ ಬದಲಾವಣೆಗಳನ್ನು ತೋರಿಸುವ ಟ್ರೆಂಡ್ ಚಾರ್ಟ್‌ಗಳು
  • ಸಂದರ್ಭಕ್ಕಾಗಿ ಕ್ಲಿನಿಕಲ್ ಉಲ್ಲೇಖ ಶ್ರೇಣಿಗಳು
  • ದತ್ತಾಂಶದ ಮೂಲ ಮಾಹಿತಿ (ಆಪಲ್ ವಾಚ್, ಐಫೋನ್, ಇತ್ಯಾದಿ)

ದತ್ತಾಂಶವನ್ನು PDF ಆಗಿ ರಫ್ತು ಮಾಡಿ ಮತ್ತು ನಿಮ್ಮ ನೇಮಕಾತಿಗೆ ಮೊದಲು ನಿಮ್ಮ ವೈದ್ಯರಿಗೆ ಇಮೇಲ್ ಮಾಡಿ ಅಥವಾ ಮುದ್ರಿತ ಪ್ರತಿಯನ್ನು ತನ್ನಿ. ಈ ಸ್ವರೂಪವನ್ನು ವೈದ್ಯಕೀಯ ಸಮಾಲೋಚನೆಗಳಿಗಾಗಿ ಸ್ಪಷ್ಟವಾಗಿ ಮತ್ತು ವೃತ್ತಿಪರವಾಗಿ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ.

XML ಮತ್ತು CSV ಆರೋಗ್ಯ ದತ್ತಾಂಶ ರಫ್ತು ನಡುವಿನ ವ್ಯತ್ಯಾಸವೇನು?

ಆಪಲ್‌ನ ಸ್ಥಳೀಯ ಆರೋಗ್ಯ ದತ್ತಾಂಶ ರಫ್ತು XML ಫೈಲ್ ಅನ್ನು ಒದಗಿಸುತ್ತದೆ, ಇದು ಸಮಗ್ರವಾಗಿದೆ ಆದರೆ ಬಳಸಲು ಕಷ್ಟ:

ವೈಶಿಷ್ಟ್ಯಆಪಲ್‌ನ XML ರಫ್ತುಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ CSV ರಫ್ತು
ಸ್ವರೂಪಸಂಕೀರ್ಣ XML ರಚನೆಸರಳ ಸ್ಪ್ರೆಡ್‌ಶೀಟ್ ಸ್ವರೂಪ
ಓದಬಲ್ಲದುತಾಂತ್ರಿಕ ಜ್ಞಾನ ಅಗತ್ಯಎಕ್ಸೆಲ್/ಶೀಟ್ಸ್‌ನಲ್ಲಿ ನೇರವಾಗಿ ತೆರೆಯುತ್ತದೆ
ಆಯ್ದ ರಫ್ತುಎಲ್ಲವೂ ಅಥವಾ ಏನೂ ಇಲ್ಲನಿರ್ದಿಷ್ಟ ಮೆಟ್ರಿಕ್‌ಗಳನ್ನು ಆರಿಸಿ
ದಿನಾಂಕ ಶ್ರೇಣಿಗಳುಸಂಪೂರ್ಣ ಇತಿಹಾಸ ಮಾತ್ರಕಸ್ಟಮ್ ದಿನಾಂಕ ಶ್ರೇಣಿಗಳು
ವೈದ್ಯಕೀಯ ಬಳಕೆವೈದ್ಯರಿಗೆ ಅಷ್ಟಾಗಿ ಸೂಕ್ತವಲ್ಲವೃತ್ತಿಪರ PDF ವರದಿಗಳು

CSV ಸ್ವರೂಪವು ದತ್ತಾಂಶ ವಿಶ್ಲೇಷಣೆಗೆ ಸೂಕ್ತವಾಗಿದೆ, ಆದರೆ ನಮ್ಮ PDF ವರದಿಗಳು ವೈದ್ಯರೊಂದಿಗೆ ಹಂಚಿಕೊಳ್ಳಲು ಉತ್ತಮವಾಗಿವೆ.

ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ಬಳಸುವಾಗ ನನ್ನ ಆರೋಗ್ಯ ದತ್ತಾಂಶವು ಗೌಪ್ಯವಾಗಿರುತ್ತದೆಯೇ?

ಖಂಡಿತವಾಗಿ. ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ಅನ್ನು ಗೌಪ್ಯತೆಗೆ ಮೊದಲ ಆದ್ಯತೆ ನೀಡಿ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ:

  • 100% ಸ್ಥಳೀಯ ಪ್ರಕ್ರಿಯೆ — ಎಲ್ಲಾ ದತ್ತಾಂಶ ವಿಶ್ಲೇಷಣೆ ನಿಮ್ಮ ಸಾಧನದಲ್ಲಿ ನಡೆಯುತ್ತದೆ
  • ಖಾತೆಯ ಅಗತ್ಯವಿಲ್ಲ — ನೋಂದಣಿ ಇಲ್ಲದೆ ತಕ್ಷಣವೇ ಬಳಸಲು ಪ್ರಾರಂಭಿಸಿ
  • ಯಾವುದೇ ಕ್ಲೌಡ್ ಅಪ್‌ಲೋಡ್‌ಗಳಿಲ್ಲ — ನಿಮ್ಮ ಆರೋಗ್ಯ ದತ್ತಾಂಶವು ನಿಮ್ಮ ಐಫೋನ್ ಅನ್ನು ಬಿಡುವುದಿಲ್ಲ
  • ಯಾವುದೇ ಟ್ರ್ಯಾಕಿಂಗ್ ಇಲ್ಲ — ನಾವು ಅನಾಲಿಟಿಕ್ಸ್ SDKಗಳನ್ನು ಬಳಸುವುದಿಲ್ಲ ಅಥವಾ ಬಳಕೆಯ ದತ್ತಾಂಶವನ್ನು ಸಂಗ್ರಹಿಸುವುದಿಲ್ಲ
  • GDPR ಅನುಸರಣೆ — ಗೌಪ್ಯತೆ ನಿಯಮಗಳ ಸಂಪೂರ್ಣ ಅನುಸರಣೆ

ನಿಮ್ಮ ದತ್ತಾಂಶವನ್ನು ಯಾವಾಗ ಮತ್ತು ಹೇಗೆ ರಫ್ತು ಮಾಡಬೇಕೆಂದು ನೀವು ನಿಯಂತ್ರಿಸುತ್ತೀರಿ. ರಫ್ತುಗಳನ್ನು ನಿಮ್ಮ ಸಾಧನದಲ್ಲಿ ಸ್ಥಳೀಯವಾಗಿ ಉಳಿಸಲಾಗುತ್ತದೆ ಮತ್ತು ನೀವು ಹಂಚಿಕೊಳ್ಳಲು ಆಯ್ಕೆ ಮಾಡಿದಾಗ ಮಾತ್ರ ಹಂಚಿಕೊಳ್ಳಲಾಗುತ್ತದೆ.

ಆಪಲ್ ಹೆಲ್ತ್ XML ಅನ್ನು PDF ಗೆ ಪರಿವರ್ತಿಸುವುದು ಹೇಗೆ?

ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ನಿಮ್ಮ HealthKit ದತ್ತಾಂಶವನ್ನು ವೃತ್ತಿಪರ PDF ವರದಿಗಳಾಗಿ ಪರಿವರ್ತಿಸುತ್ತದೆ:

  1. ನಿಮ್ಮ ಐಫೋನ್‌ನಲ್ಲಿ ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ತೆರೆಯಿರಿ
  2. ನೀವು ಸೇರಿಸಲು ಬಯಸುವ ಆರೋಗ್ಯ ಮೆಟ್ರಿಕ್‌ಗಳನ್ನು ಆಯ್ಕೆಮಾಡಿ
  3. ನಿಮ್ಮ ದಿನಾಂಕ ಶ್ರೇಣಿಯನ್ನು ಆರಿಸಿ (ಕಳೆದ ವಾರ, ತಿಂಗಳು, ವರ್ಷ ಅಥವಾ ಕಸ್ಟಮ್)
  4. Export ಒತ್ತಿ ಮತ್ತು "PDF Report" ಅನ್ನು ಆಯ್ಕೆಮಾಡಿ
  5. ಅಪ್ಲಿಕೇಶನ್ ಚಾರ್ಟ್‌ಗಳು ಮತ್ತು ಸಂದರ್ಭದೊಂದಿಗೆ ಫಾರ್ಮ್ಯಾಟ್ ಮಾಡಿದ PDF ಅನ್ನು ರಚಿಸುತ್ತದೆ
  6. ಇಮೇಲ್, AirDrop ಮೂಲಕ ಹಂಚಿಕೊಳ್ಳಿ ಅಥವಾ ಫೈಲ್‌ಗಳಿಗೆ ಉಳಿಸಿ

ಹಸ್ತಚಾಲಿತವಾಗಿ XML ರಫ್ತು ಮಾಡಿ ಪರಿವರ್ತಿಸುವ ಅಗತ್ಯವಿಲ್ಲ. ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ನೇರವಾಗಿ HealthKit ನಿಂದ ಓದುತ್ತದೆ ಮತ್ತು ವೈದ್ಯಕೀಯ ಸಮಾಲೋಚನೆಗಳಿಗೆ ಸೂಕ್ತವಾದ ಸ್ಪಷ್ಟ, ಓದಬಲ್ಲ PDFಗಳನ್ನು ರಚಿಸುತ್ತದೆ.

ನಾನು HealthKit ನಿಂದ ಯಾವ ಆರೋಗ್ಯ ಮೆಟ್ರಿಕ್‌ಗಳನ್ನು ರಫ್ತು ಮಾಡಬಹುದು?

ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ಎಲ್ಲಾ ಪ್ರಮುಖ HealthKit ದತ್ತಾಂಶ ಪ್ರಕಾರಗಳನ್ನು ರಫ್ತು ಮಾಡುವುದನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ:

ಚಲನೆ ಮತ್ತು ಫಿಟ್‌ನೆಸ್

ಹೆಜ್ಜೆಗಳು, ದೂರ, ಹತ್ತಿದ ಮಹಡಿಗಳು, ಸಕ್ರಿಯ ಶಕ್ತಿ, ವ್ಯಾಯಾಮದ ಸಮಯ, VO₂ Max, ವರ್ಕ್‌ಔಟ್‌ಗಳು

ಹೃದಯ ಮತ್ತು ರಕ್ತಪರಿಚಲನೆ

ಹೃದಯ ಬಡಿತ, ವಿಶ್ರಾಂತಿ ಹೃದಯ ಬಡಿತ, ಹೃದಯ ಬಡಿತದ ವ್ಯತ್ಯಾಸ (HRV), ರಕ್ತದೊತ್ತಡ, ECG ದತ್ತಾಂಶ, ಅನಿಯಮಿತ ಬಡಿತದ ಅಧಿಸೂಚನೆಗಳು

ನಿದ್ರೆ ಮತ್ತು ಚೇತರಿಕೆ

ನಿದ್ರೆಯ ಅವಧಿ, ನಿದ್ರೆಯ ಹಂತಗಳು (REM, ಕೋರ್, ಡೀಪ್), ಹಾಸಿಗೆಯಲ್ಲಿ ಕಳೆದ ಸಮಯ, ಉಸಿರಾಟದ ದರ

ದೇಹದ ಅಳತೆಗಳು

ತೂಕ, BMI, ದೇಹದ ಕೊಬ್ಬಿನ ಪ್ರತಿಶತ, ಲೀನ್ ಬಾಡಿ ಮಾಸ್, ಎತ್ತರ, ಸೊಂಟದ ಸುತ್ತಳತೆ

ಪೋಷಣೆ ಮತ್ತು ಜಲಸಂಚಯನ

ಶಕ್ತಿ ಸೇವನೆ, ಮ್ಯಾಕ್ರೋನ್ಯೂಟ್ರಿಯೆಂಟ್‌ಗಳು, ಮೈಕ್ರೋನ್ಯೂಟ್ರಿಯೆಂಟ್‌ಗಳು, ನೀರು ಸೇವನೆ

ಕ್ಲಿನಿಕಲ್ ಮತ್ತು ಲ್ಯಾಬ್ಸ್

ರಕ್ತದ ಗ್ಲೂಕೋಸ್, HbA1c, ಇನ್ಸುಲಿನ್ ವಿತರಣೆ, ರಕ್ತದ ಮದ್ಯದ ಅಂಶ, ಆಮ್ಲಜನಕದ ಶುದ್ಧತ್ವ (SpO2)

ಎಲ್ಲಾ ಮೆಟ್ರಿಕ್‌ಗಳನ್ನು ಪ್ರತ್ಯೇಕವಾಗಿ ಅಥವಾ ಒಟ್ಟಾಗಿ CSV, JSON ಅಥವಾ PDF ಸ್ವರೂಪದಲ್ಲಿ ರಫ್ತು ಮಾಡಬಹುದು.

ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ಬಳಸಲು ನನಗೆ ಆಪಲ್ ವಾಚ್ ಅಗತ್ಯವಿದೆಯೇ?

ಇಲ್ಲ, ನಿಮಗೆ ಆಪಲ್ ವಾಚ್ ಅಗತ್ಯವಿಲ್ಲ. ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ನಿಮ್ಮ ಆಪಲ್ ಹೆಲ್ತ್ ಅಪ್ಲಿಕೇಶನ್‌ನಲ್ಲಿರುವ ಯಾವುದೇ ದತ್ತಾಂಶದೊಂದಿಗೆ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ:

  • ಐಫೋನ್ ಸಂವೇದಕ ದತ್ತಾಂಶ (ಹೆಜ್ಜೆಗಳು, ದೂರ)
  • ಆಪಲ್ ವಾಚ್ ಅಳತೆಗಳು (ನೀವು ಹೊಂದಿದ್ದರೆ)
  • ಮೂರನೇ ವ್ಯಕ್ತಿಯ ಸಂಪರ್ಕಿತ ಸಾಧನಗಳು (ಸ್ಮಾರ್ಟ್ ಮಾಪಕಗಳು, ಬಿಪಿ ಮಾನಿಟರ್‌ಗಳು, ಗ್ಲೂಕೋಸ್ ಮೀಟರ್‌ಗಳು)
  • ಹೆಲ್ತ್ ಅಪ್ಲಿಕೇಶನ್‌ಗೆ ನೀವು ಸೇರಿಸಿದ ಹಸ್ತಚಾಲಿತ ನಮೂದುಗಳು
  • ಇತರ ಆರೋಗ್ಯ ಮತ್ತು ಫಿಟ್‌ನೆಸ್ ಅಪ್ಲಿಕೇಶನ್‌ಗಳ ದತ್ತಾಂಶ

ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ಬೆಲೆ ಎಷ್ಟು?

ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ 7-ದಿನಗಳ ಉಚಿತ ಪ್ರಯೋಗವನ್ನು ನೀಡುತ್ತದೆ. ಪ್ರಯೋಗದ ನಂತರ:

  • ನಿಮ್ಮ ಎಲ್ಲಾ ಆರೋಗ್ಯ ದತ್ತಾಂಶವನ್ನು ವೀಕ್ಷಿಸಿ ಮತ್ತು ವಿಶ್ಲೇಷಿಸಿ
  • CSV, JSON ಮತ್ತು PDF ಸ್ವರೂಪಗಳಲ್ಲಿ ಅನಿಯಮಿತ ರಫ್ತುಗಳು
  • ವೈಜ್ಞಾನಿಕ ಸಂದರ್ಭ ಮತ್ತು ಉಲ್ಲೇಖ ಶ್ರೇಣಿಗಳಿಗೆ ಪ್ರವೇಶ
  • ಹೊಸ ವೈಶಿಷ್ಟ್ಯಗಳೊಂದಿಗೆ ನಿಯಮಿತ ನವೀಕರಣಗಳು

ಖಾತೆಯ ಅಗತ್ಯವಿಲ್ಲ, ಮತ್ತು ನೀವು ಯಾವಾಗ ಬೇಕಾದರೂ ರದ್ದುಗೊಳಿಸಬಹುದು. ನಿಮ್ಮ ದತ್ತಾಂಶ ಯಾವಾಗಲೂ ನಿಮ್ಮ ಸಾಧನದಲ್ಲಿಯೇ ಉಳಿಯುತ್ತದೆ.

ಸಂಶೋಧನಾ ಉದ್ದೇಶಗಳಿಗಾಗಿ ನಾನು ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ಬಳಸಬಹುದೇ?

ಹೌದು! ಕಸ್ಟಮ್ ವಿಶ್ಲೇಷಣೆಗಾಗಿ JSON ಮತ್ತು CSV ರಫ್ತುಗಳನ್ನು ಬಳಸಬಹುದು. ಅನೇಕ ಬಳಕೆದಾರರು ವೈಯಕ್ತಿಕ ಸಂಶೋಧನೆ ಅಥವಾ ಶೈಕ್ಷಣಿಕ ಯೋಜನೆಗಳಿಗಾಗಿ ದತ್ತಾಂಶವನ್ನು ರಫ್ತು ಮಾಡುತ್ತಾರೆ.

ಇನ್ನೂ ಪ್ರಶ್ನೆಗಳಿವೆಯೇ?

ಹೆಲ್ತ್ ಅನಾಲಿಟಿಕ್ಸ್ ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ ಮತ್ತು ನಿಮ್ಮ ಆರೋಗ್ಯ ದತ್ತಾಂಶವನ್ನು ರಫ್ತು ಮಾಡುವುದು ಎಷ್ಟು ಸುಲಭ ಎಂದು ನೋಡಿ.
