如何使用 Health Analytics

Health Analytics 帮助您充分利用 Apple 健康数据的真实场景

📋 为医生预约准备健康数据

生成医疗保健提供者实际可用的专业医疗报告。

问题

您想与通过医生分享您的健康数据，但在预约期间向他们展示手机屏幕并不切实际。Apple 的原生 XML 导出对医疗专业人员来说技术性太强。

解决方案

Health Analytics 创建干净、专业的 PDF 报告，包含：

  • 与您的咨询相关的选定指标
  • 显示随时间变化的趋势图表
  • 用于背景参考的临床参考范围
  • 数据来源信息（设备来源）

常见场景

  • 心脏病学： 导出心率、HRV、血压、ECG 数据用于心脏健康评估
  • 睡眠医学： 分享睡眠阶段、时长和呼吸指标
  • 内分泌学： 提供血糖、胰岛素和营养数据用于糖尿病管理
  • 运动医学： 显示 VO₂ Max、锻炼时间和恢复指标

快速步骤

  1. 打开 Health Analytics
  2. 选择相关的健康指标
  3. 选择日期范围（例如，过去 3 个月）
  4. 导出为 PDF
  5. 通过电子邮件发送给医生或携带打印副本

专业提示： 在预约前发送报告，以便您的医生可以提前查看。

📊 在电子表格中分析 Apple 健康趋势

将您的 HealthKit 数据导出为 CSV 以进行自定义分析和可视化。

问题

Apple 健康显示基本图表，但您想进行更深入的分析：指标之间的相关性、自定义时间段或与其他数据源结合。

解决方案

导出为 CSV 并在 Excel、Google Sheets 或 Numbers 中打开以：

  • 创建自定义图表和数据透视表
  • 计算相关性（例如，睡眠质量与 HRV）
  • 结合营养或训练日志
  • 追踪实现健康目标的进度

流行的分析用例

  • 减肥追踪： 关联随时间变化的体重、活动和营养
  • 训练优化： 分析锻炼强度、恢复和表现趋势
  • 睡眠改善： 识别睡眠质量、HRV 和日常习惯之间的模式
  • 慢性病管理： 追踪症状以及生命体征和药物

您可以导出什么

  • 步数、距离、爬楼层数
  • 心率、HRV、血压
  • 睡眠阶段和时长
  • 体重、BMI、身体成分
  • 营养和水合
  • 锻炼和活动能量
  • VO₂ Max 和心肺适能
  • 血糖和胰岛素

格式： 带有时间戳、数值和单位的干净 CSV，可直接用于分析。

💾 备份您的 Apple 健康数据

以可用格式创建健康历史记录的安全、便携备份。

问题

您的健康数据很有价值且不可替代。如果您丢失了 iPhone 或更换设备，您希望确保以您可以实际访问的格式保存您的健康历史记录。

解决方案

定期导出创建备份，这些备份是：

  • 可移植： CSV 和 JSON 适用于任何平台
  • 面向未来： 不锁定在 Apple 生态系统中
  • 可读： 无需特殊软件
  • 选择性： 仅导出对您重要的内容

备份策略

  • 每月完整备份： 将所有指标导出为 CSV 到云存储
  • 季度 PDF 摘要： 为个人记录创建易读的报告
  • 设备升级前： 在更换手机前完成导出
  • 年度 JSON 归档： 适合长期存储的开发者友好格式

为什么备份很重要

  • 一处保存多年的健康历史
  • 防止设备丢失或故障
  • 切换平台的自由
  • 与未来的医疗保健提供者分享
  • 个人健康研究和追踪

隐私说明： 所有导出都在您的设备本地。您控制备份的存储位置。

🔬 用于研究和自我追踪的健康数据

为个人研究、学术研究或量化自我项目导出结构化数据。

问题

您正在参与一项健康研究，进行个人研究，或者想用专门的工具分析您的数据。您需要研究人员可以使用的干净、结构化的数据格式。

解决方案

Health Analytics 提供研究级导出：

  • JSON 格式： 用于 Python、R 或统计软件的结构化数据
  • CSV 格式： 导入 SPSS、SAS 或 Excel 进行分析
  • 数据来源： 了解每次测量记录的设备
  • 精确时间戳： 精确到秒的时间序列分析

研究应用

  • 学术研究： 将您的数据贡献给健康研究项目
  • N-of-1 试验： 带有干预和结果的个人实验
  • 量化自我： 深入研究个人健康模式
  • 机器学习： 基于您自己的健康数据训练模型

数据质量功能

  • 源设备识别
  • 测量精度元数据
  • 重复检测和处理
  • 自定义日期范围选择
  • 特定指标过滤
  • 包含单位和换算

对于开发者： JSON 导出包含用于高级分析的完整 HealthKit 元数据。
