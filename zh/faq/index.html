常见问题解答

关于导出和分析您的 Apple 健康数据的常见问题

如何将我的 Apple 健康数据导出为 CSV？

使用 Health Analytics，将您的 Apple 健康数据导出为 CSV 非常简单：

  1. 从 App Store 下载 Health Analytics
  2. 打开应用程序并授予 HealthKit 权限
  3. 导航到您要导出的指标
  4. 点击导出按钮并选择“CSV”格式
  5. 选择您的日期范围和特定数据点
  6. 分享或保存 CSV 文件

CSV 文件可以在 Excel、Google Sheets、Numbers 或任何电子表格应用程序中打开。与 Apple 的原生 XML 导出不同，我们的 CSV 文件可立即用于分析。

我可以与医生分享我的 HealthKit 数据吗？

可以！Health Analytics 让您可以轻松地与医疗保健提供者分享您的健康数据。您可以生成包含以下内容的专业 PDF 报告：

  • 与您的预约相关的选定健康指标
  • 显示随时间变化的趋势图表
  • 用于背景参考的临床参考范围
  • 数据来源信息（Apple Watch、iPhone 等）

只需将数据导出为 PDF 并在预约前通过电子邮件发送给您的医生，或携带打印副本。该格式旨在为医疗咨询提供清晰和专业的信息。

XML 和 CSV 健康数据导出有什么区别？

Apple 的原生健康数据导出提供了一个 XML 文件，虽然全面但难以使用：

功能Apple 的 XML 导出Health Analytics CSV 导出
格式复杂的 XML 结构简单的电子表格格式
可读性需要技术知识直接在 Excel/Sheets 中打开
选择性导出全有或全无选择特定指标
日期范围仅限整个历史记录自定义日期范围
医疗用途不适合医生查看专业的 PDF 报告

CSV 格式非常适合数据分析，而我们的 PDF 报告非常适合与医疗保健提供者分享。

使用 Health Analytics 时我的健康数据私密吗？

绝对私密。Health Analytics 将隐私作为首要任务：

  • 100% 本地处理 — 所有数据分析都在您的设备上进行
  • 无需账户 — 无需注册即可立即开始使用
  • 无云端上传 — 您的健康数据从未离开您的 iPhone
  • 无追踪 — 我们不使用分析 SDK 或收集使用数据
  • GDPR 合规 — 完全符合隐私法规

您控制何时以及如何导出数据。导出仅保存在您的设备本地，并且仅在您选择分享时才会被分享。

如何将 Apple Health XML 转换为 PDF？

Health Analytics 自动将您的 HealthKit 数据转换为专业的 PDF 报告：

  1. 在您的 iPhone 上打开 Health Analytics
  2. 选择您要包含的健康指标
  3. 选择您的日期范围（上周、上月、去年或自定义）
  4. 点击导出并选择“PDF 报告”
  5. 应用程序会生成带有图表和背景信息的格式化 PDF
  6. 通过电子邮件、AirDrop 分享或保存到文件

无需手动导出 XML 并进行转换。Health Analytics 直接从 HealthKit 读取数据，并创建适合医疗预约的清晰、易读的 PDF。

我可以从 HealthKit 导出哪些健康指标？

Health Analytics 支持导出所有主要的 HealthKit 数据类型，按类别组织：

运动与健身

步数、距离、爬楼层数、活动能量、锻炼时间、VO₂ Max、锻炼

心脏与循环

心率、静息心率、心率变异性 (HRV)、血压、ECG 数据、心律不齐通知

睡眠与恢复

睡眠时长、睡眠阶段（REM、核心、深度）、卧床时间、呼吸频率

身体测量

体重、BMI、体脂率、瘦体重、身高、腰围

营养与水合

能量摄入、宏量营养素（碳水化合物、蛋白质、脂肪）、微量营养素、水分消耗

临床与实验室

血糖、HbA1c、胰岛素输送、血液酒精含量、血氧饱和度

所有指标均可以 CSV、JSON 或 PDF 格式单独或组合导出。

我需要 Apple Watch 才能使用 Health Analytics 吗？

不，您不需要 Apple Watch。Health Analytics 适用于 Apple 健康应用中的任何数据，包括：

  • iPhone 传感器数据（步数、楼层、距离）
  • Apple Watch 测量数据（如果您有）
  • 第三方连接设备（智能秤、血压计、血糖仪）
  • 您添加到健康应用的手动条目
  • 来自其他健康和健身应用的数据

您在 Apple 健康中的数据越多，您获得的见解就越多，但 Apple Watch 并不是必需的。

我可以将 Health Analytics 用于研究目的吗？

是的！Health Analytics 非常适合个人健康研究和自我追踪：

  • JSON 导出 — 用于分析脚本的结构化数据格式
  • CSV 导出 — 导入 R、Python、SPSS 或统计工具
  • 自定义日期范围 — 导出用于研究的特定时间段
  • 数据来源 — 了解每次测量记录的设备
  • 科学引用 — 以同行评审研究为后盾的参考范围

许多用户导出他们的数据用于个人研究、学术项目或与研究分享。
