자주 묻는 질문

Apple 건강 데이터 내보내기 및 분석에 관한 일반적인 질문들입니다.

Apple 건강 데이터를 CSV로 어떻게 내보내나요?

Health Analytics를 사용하면 Apple 건강 데이터를 CSV로 내보내는 과정이 매우 간단합니다:

  1. App Store에서 Health Analytics를 다운로드합니다.
  2. 앱을 실행하고 HealthKit 접근 권한을 승인합니다.
  3. 내보내고 싶은 지표 화면으로 이동합니다.
  4. '내보내기(Export)' 버튼을 누르고 "CSV" 형식을 선택합니다.
  5. 날짜 범위와 특정 데이터 포인트를 선택합니다.
  6. CSV 파일을 공유하거나 저장합니다.

생성된 CSV 파일은 Excel, Google Sheets, Numbers 등 모든 스프레드시트 앱에서 열 수 있습니다. Apple의 기본 XML 내보내기와 달리, 저희가 제공하는 CSV 파일은 즉시 분석에 활용할 수 있습니다.

제 건강 데이터를 의사에게 공유할 수 있나요?

네! Health Analytics는 건강 데이터를 의료 제공자와 쉽게 공유할 수 있도록 돕습니다. 다음과 같은 내용이 포함된 전문적인 PDF 보고서를 생성할 수 있습니다:

  • 상담에 필요한 특정 건강 지표 선택
  • 시간 경과에 따른 변화를 보여주는 추세 차트
  • 이해를 돕기 위한 임상 참조 범위
  • 데이터 출처 정보 (Apple Watch, iPhone 등)

데이터를 PDF로 내보내어 진료 전에 의사에게 이메일로 보내거나 인쇄하여 지참하십시오. 이 형식은 의료 상담을 위해 명확하고 전문적으로 설계되었습니다.

XML과 CSV 건강 데이터 내보내기의 차이점은 무엇인가요?

Apple의 기본 건강 데이터 내보내기는 XML 파일을 제공하는데, 이는 포괄적이지만 사용하기가 매우 어렵습니다:

기능Apple 기본 XML 내보내기Health Analytics CSV 내보내기
형식복잡한 XML 구조단순한 스프레드시트 형식
가독성기술적 지식 필요Excel/Sheets에서 즉시 확인 가능
선택적 내보내기전체 데이터만 가능특정 지표만 선택 가능
날짜 범위전체 기록만 가능사용자 정의 기간 설정 가능
의료용 활용의료진이 확인하기 어려움전문가 수준의 PDF 보고서

CSV 형식은 데이터 분석에 이상적이며, PDF 보고서는 의료진과 공유하기에 완벽합니다.

Health Analytics를 사용할 때 제 건강 데이터는 안전한가요?

물론입니다. Health Analytics는 개인정보 보호를 최우선으로 설계되었습니다:

  • 100% 로컬 처리 — 모든 데이터 분석은 사용자의 기기 내에서 이루어집니다.
  • 계정 생성 불필요 — 별도의 등록 없이 즉시 사용을 시작할 수 있습니다.
  • 클라우드 업로드 없음 — 귀하의 건강 데이터는 절대 iPhone을 떠나지 않습니다.
  • 추적 없음 — 분석 SDK를 사용하거나 사용 데이터를 수집하지 않습니다.
  • GDPR 준수 — 개인정보 보호 규정을 엄격히 준수합니다.

귀하는 데이터를 언제, 어떻게 내보낼지 완전히 통제할 수 있습니다. 내보낸 데이터는 기기에 로컬로 저장되며 귀하가 선택한 경우에만 공유됩니다.

Apple 건강 XML을 PDF로 어떻게 변환하나요?

Health Analytics는 HealthKit 데이터를 전문가 수준의 PDF 보고서로 자동 변환합니다:

  1. iPhone에서 Health Analytics를 엽니다.
  2. 포함하고 싶은 건강 지표를 선택합니다.
  3. 날짜 범위(지난주, 지난달, 지난해 또는 사용자 지정)를 선택합니다.
  4. '내보내기'를 누르고 "PDF 보고서"를 선택합니다.
  5. 앱이 차트와 맥락이 포함된 정돈된 PDF를 생성합니다.
  6. 이메일, AirDrop으로 공유하거나 파일 앱에 저장합니다.

XML을 수동으로 내보내어 변환할 필요가 없습니다. Health Analytics는 HealthKit에서 직접 데이터를 읽어 진료 시 바로 사용할 수 있는 깨끗하고 읽기 쉬운 PDF를 만듭니다.

HealthKit에서 어떤 건강 지표를 내보낼 수 있나요?

Health Analytics는 카테고리별로 구성된 모든 주요 HealthKit 데이터 유형의 내보내기를 지원합니다:

운동 및 피트니스

걸음 수, 거리, 계단 오른 층수, 활동 에너지, 운동 시간, 최대 산소 섭취량(VO₂ Max), 운동 기록

심장 및 순환

심박수, 안정 시 심박수, 심박 변이도(HRV), 혈압, 심전도 데이터, 불규칙한 리듬 알림

수면 및 회복

수면 시간, 수면 단계(REM, 코어, 깊은 수면), 침대에 누워 있는 시간, 호흡률

신체 측정

체중, BMI, 체지방률, 제지방량, 키, 허리둘레

영양 및 수분 섭취

에너지 섭취량, 다량 영양소(탄수화물, 단백질, 지방), 미량 영양소, 수분 섭취량

진료 및 검사 기록

혈당, 당화혈색소(HbA1c), 인슐린 투여량, 혈중 알코올 농도, 산소 포화도

모든 지표는 개별적으로 또는 조합하여 CSV, JSON 또는 PDF 형식으로 내보낼 수 있습니다.

Health Analytics를 사용하려면 Apple Watch가 필요한가요?

아닙니다. Apple Watch가 없어도 사용 가능합니다. Health Analytics는 다음을 포함하여 Apple 건강 앱에 있는 모든 데이터와 연동됩니다:

  • iPhone 센서 데이터 (걸음 수, 계단 층수, 거리)
  • Apple Watch 측정 데이터 (착용 중인 경우)
  • 제3자 연결 기기 (스마트 체중계, 혈압계, 혈당계 등)
  • 건강 앱에 직접 수동으로 입력한 데이터
  • 다른 건강 및 피트니스 앱의 데이터

Apple 건강에 더 많은 데이터가 있을수록 더 풍부한 통찰력을 얻을 수 있지만, Apple Watch가 필수 조건은 아닙니다.

Health Analytics 이용 요금은 얼마인가요?

Health Analytics는 모든 기능을 미리 경험해 볼 수 있도록 7일 무료 평가판을 제공합니다. 평가판 이후에는 다음을 이용할 수 있습니다:

  • 모든 건강 데이터 확인 및 분석
  • CSV, JSON, PDF 형식의 무제한 내보내기
  • 과학적 맥락 및 참조 범위 접근
  • 새로운 기능 및 지표에 대한 정기 업데이트

계정이 필요 없으며 언제든지 취소할 수 있습니다. 당신의 데이터는 항상 기기에 머뭅니다.

Health Analytics를 연구 목적으로 사용할 수 있나요?

네! Health Analytics는 개인 건강 연구 및 자가 추적에 이상적입니다:

  • JSON 내보내기 — 분석 스크립트를 위한 구조화된 데이터 형식
  • CSV 내보내기 — R, Python, SPSS 또는 통계 도구로 데이터 임포트
  • 사용자 정의 날짜 범위 — 연구 목적에 맞는 특정 기간 데이터 내보내기
  • 데이터 출처 명시 — 각 측정을 기록한 기기 확인
  • 과학적 인용 정보 — 피어 리뷰 연구를 바탕으로 한 참조 범위

많은 사용자들이 개인 연구, 학술 프로젝트를 위해 또는 연구 프로젝트와 데이터를 공유하기 위해 내보내기 기능을 활용하고 있습니다.

더 궁금한 점이 있으신가요?

Health Analytics를 다운로드하여 건강 데이터를 내보내고 분석하는 것이 얼마나 쉬운지 직접 확인해 보세요.
